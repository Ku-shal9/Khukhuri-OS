Command Overview

Core shell commands available after boot. Organized for quick lookup while testing features in-browser.

Command Purpose Category
sahayog Shows the shell help menu and command hints. General
safa Clears the terminal screen. General
samaya Displays current time from RTC. System
jhalak Shows compact FAT and file summary. Filesystem
fatlist Lists all files with size and start block. Filesystem
create / write / read Create files, write content, and read back data. Filesystem
proclist / thlist / sched / policy Inspect process/thread state and scheduler policy. Process
mkproc / mkthr Spawn demo processes and threads for scheduler tests. Process
calc Runs shell math utility. Utility

Kernel Features

• Custom boot banner and shell flow
• Keyboard, timer, mouse, and filesystem drivers
• Interrupt + IDT initialization sequence
• Lightweight kernel loop for stability

How Browser Boot Works

• `libv86.js` runs an x86 emulator in JavaScript
• BIOS and VGA BIOS are loaded from CDN
• Your local `build/os.iso` is mounted as CD-ROM
• System starts automatically once assets load

Keyboard Help

Click inside the emulator screen first.
Ctrl + Option (Mac) or Ctrl + Alt (Windows/Linux) releases mouse capture.
On mobile, tap screen to open keyboard input.
Use your shell commands as usual after boot.
